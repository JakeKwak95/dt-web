DT_CatholicHospital WebGL 실행 방법
=================================

이 폴더는 Unity WebGL 빌드와 테스트용 mock-canvas.html을 포함합니다.
WebGL 빌드는 html 파일을 더블클릭해서 직접 여는 방식(file://)으로는 정상 실행되지 않을 수 있습니다.
반드시 이 폴더를 기준으로 로컬 웹 서버를 실행한 뒤 브라우저에서 접속해 주세요.


1. 압축 해제
-----------

1) 전달받은 zip 파일을 원하는 위치에 압축 해제합니다.
2) 압축을 푼 폴더 안에 아래 항목들이 있는지 확인합니다.

- Build
- StreamingAssets
- TemplateData
- index.html
- mock-canvas.html
- HowToUse.txt


2. Python이 설치되어 있는 경우
-----------------------------

1) 압축을 푼 폴더를 엽니다.
2) 주소창에 cmd를 입력하거나, 폴더 안의 빈 공간에서 터미널/명령 프롬프트를 엽니다.
3) 아래 명령어를 실행합니다.

python -m http.server 8080

만약 위 명령어가 동작하지 않으면 아래 명령어도 시도해 보세요.

py -m http.server 8080

4) 브라우저에서 아래 주소를 엽니다.

http://localhost:8080/mock-canvas.html


3. Python이 없는 경우
--------------------

아래 방법 중 하나를 사용하면 됩니다.


방법 A. Visual Studio Code + Live Server 사용

1) Visual Studio Code를 설치합니다.
2) Visual Studio Code에서 이 폴더를 엽니다.
3) Extensions에서 "Live Server"를 설치합니다.
4) mock-canvas.html을 우클릭한 뒤 "Open with Live Server"를 선택합니다.
5) 브라우저에 열린 페이지에서 동작을 확인합니다.


방법 B. Node.js가 설치되어 있는 경우

1) 압축을 푼 폴더에서 터미널/명령 프롬프트를 엽니다.
2) 아래 명령어를 실행합니다.

npx http-server -p 8080

3) 브라우저에서 아래 주소를 엽니다.

http://localhost:8080/mock-canvas.html


방법 C. Python을 설치해서 실행

1) https://www.python.org/downloads/ 에서 Python을 설치합니다.
2) 설치 화면에서 "Add python.exe to PATH" 옵션을 체크합니다.
3) 설치 후 새 명령 프롬프트를 열고 아래 명령어를 실행합니다.

python -m http.server 8080

4) 브라우저에서 아래 주소를 엽니다.

http://localhost:8080/mock-canvas.html


4. 사용 방법
-----------

페이지가 정상적으로 열리면 왼쪽 패널에 층 선택 버튼이 표시됩니다.
Unity 로딩이 완료되면 버튼이 활성화됩니다.

버튼을 누르면 WebGL 화면의 Unity 앱으로 해당 층 정보가 전달됩니다.


5. StudioScene 조작 방법
-----------------------

StudioScene에서는 화면 오른쪽의 오브젝트 목록에서 배치할 오브젝트를 선택할 수 있습니다.
처음 시작했을 때는 선택된 오브젝트가 없습니다.


오브젝트 선택 및 배치

1) 오른쪽 오브젝트 목록에서 배치할 오브젝트 아이콘을 클릭합니다.
2) 마우스를 바닥 위로 이동하면 배치 미리보기 오브젝트가 표시됩니다.
3) 미리보기 색상이 초록색이면 배치할 수 있는 위치입니다.
4) 미리보기 색상이 빨간색이면 이미 배치된 오브젝트와 겹치거나 배치할 수 없는 위치입니다.
5) 배치 가능한 위치에서 마우스 왼쪽 버튼을 클릭하면 오브젝트가 배치됩니다.


카메라 조작

- 마우스 오른쪽 버튼을 누른 채 위/아래로 드래그: 화면 확대/축소
- 마우스 가운데 버튼을 누른 채 드래그: 카메라 이동


배치 오브젝트 회전

- 오브젝트를 선택한 상태에서 마우스 휠 스크롤: 배치할 오브젝트 회전
- 오른쪽 오브젝트 목록 위에서 스크롤할 때는 목록 스크롤이 우선됩니다.


주의 사항

- 오브젝트가 선택되어 있어야 미리보기와 배치 기능이 동작합니다.
- 빨간색 미리보기 상태에서는 클릭해도 오브젝트가 배치되지 않습니다.
